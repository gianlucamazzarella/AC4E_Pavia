# Topic 2 Prompt Suggestions

```text
Read my project brief and draft a research design memo. Keep claims modest.
Separate what is known, assumed, and still blocked.
```

```text
Turn this research design into EARS-style requirements. Use SHALL only for
requirements that can be tested or reviewed.
```

```text
Review my data source map. Flag missing access information, unclear units,
version risks, and privacy issues.
```

```text
Create a backlog from this memo. Mark each task as parallel, sequential, or
blocked-by another task.
```
